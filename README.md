# e-Biz Technocrats Website

A modern, responsive corporate website developed for **e-Biz Technocrats** to showcase the company's services, career opportunities, company information, and contact details. The website provides a clean user experience with responsive design and interactive features.

## 🌐 Live Demo

Website link : https://e-biz-technocrats-website.vercel.app

## 📌 Features

- Responsive design for desktop, tablet, and mobile
- Light and Dark theme support
- Modern and attractive UI
- Company overview section
- Services page
- Career page
- Contact form
- Smooth navigation
- Interactive animations
- Fast loading and optimized performance

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL
